# Comprehensive Analysis and Dietary Strategies with Tableau: A College Food Choices Case Study

## Project Overview
This project analyzes college students' food choices and dietary habits using Tableau. The dashboard provides insights into eating patterns, health awareness, comfort food preferences, breakfast consumption, exercise frequency, and their relationship with student health.

## Tools Used
- Tableau Public
- CSV Dataset
- GitHub

## Dataset
- food_coded.csv

## Features
- KPI Summary Card
- Breakfast Consumption Analysis
- Eating Out vs Cooking Frequency
- Health Category by Gender
- Vitamins Consumption by Gender
- Health vs Greek Food Consumption
- People Feeling Healthy and Life Rewarded
- Weight vs Exercise Frequency
- Comfort Food Reasons
- Interactive Dashboard
- Story Presentation
